# AAARRrerg
Pirate shooter made for Circudelia launch party

Install Instructions
* Run the setup and the rest should become apparent

Controls - Best to pad these to a proper games controller until I change them
* Player 1
* Up    - W
* Down  - S
* Left  - A
* Right - D
* Fire  - SPACE

* Player 2
* Up    - Up Arrow
* Down  - Down Arrow
* Left  - Left Arrow
* Right - Right Arrow
* Fire  - ENTER

* Quit  - Esc
